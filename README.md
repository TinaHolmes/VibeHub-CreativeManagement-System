# VibeHub-CreativeManagement-System
# 🎨 Vibe Hub | 创意工作室综合管理系统

![Java](https://img.shields.io/badge/Language-Java-orange) ![MySQL](https://img.shields.io/badge/Database-MySQL-blue) ![UI](https://img.shields.io/badge/Style-Apple--Minimalism-black)

Vibe Hub 是一款专为创意工作室打造的后端管理系统，旨在通过极简的视觉交互（Apple Style）处理复杂的业务流程。

## 🌟 核心功能
* **人员管理**：基于 MVC 架构的员工信息动态维护。
* **流程审批中心**：支持请假、报销、器材调用等多种流程的实时筛选与状态更新。
* **创意空间调度**：摄影棚、会议室的预约与状态监控。
* **器材资产管理**：昂贵摄影器材的出库与归还追踪。

## 🛠️ 技术栈
* **Backend**: Java Servlet, JDBC, MySQL 8.0
* **Frontend**: JSP, CSS3 (Flexbox/Grid), JavaScript (Vanilla JS)
* **Icons**: FontAwesome 6.0
* **Architecture**: MVC Pattern

## 🚀 快速开始
1. 克隆项目：`git clone https://github.com/TinaHolmes/VibeHub.git`
2. 导入数据库脚本：执行 `sql/db_init.sql`
3. 配置 `DBUtil.java` 中的数据库连接信息
4. 部署至 Tomcat 9.0+ 运行
