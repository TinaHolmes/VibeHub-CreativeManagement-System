# VibeHub-CreativeManagement-System
A sleek, Apple-inspired Java Web management system designed for creative studios. Features include staff management, equipment scheduling, and dynamic approval workflows with a modern UI.本项目是一个基于 Web 的企业级登录模块示例，采用简洁、现代的界面设计风格，注重系统稳定性与用户体验。 适用于后台管理系统、内部信息系统等正式业务场景，可作为课程设计或企业级系统的基础模块。
